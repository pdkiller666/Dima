# Dima — Лендинг такси Дмитрия

Одностраничный сайт для водителя такси Дмитрия. Сбор отзывов клиентов, контактная информация, возможность заказать такси.

## Стек

- HTML5
- CSS3 (Flexbox/Grid, адаптивная верстка)
- JavaScript (ES6+)
- LocalStorage для хранения отзывов

## Как запустить

1. Клонируйте репозиторий:
   bash
   git clone https://github.com/your-username/dima.git
   cd dima
   
2. Откройте `index.html` в браузере.

Или используйте GitHub Pages: настройте ветку `main` в Settings → Pages.

## Функции

- Форма добавления отзыва с валидацией
- Лента отзывов (сортировка по дате, новые сверху)
- Фильтр отзывов по оценке
- Удаление собственных отзывов
- Контакты: телефон, WhatsApp, Telegram
- Адаптивный дизайн
- Бургер-меню на мобильных
- Всплывающие уведомления
