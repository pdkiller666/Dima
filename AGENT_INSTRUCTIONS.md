# Инструкции для агента

## Задача
Улучшить проект лендинга такси Дмитрия на основе фидбека ревьюера.

## Основные проблемы (из фидбека)
1. В README.md указан скриншот через placeholder.com — заменён на реальное изображение из Unsplash.
2. В CI.yml используется `|| true`, что игнорирует ошибки линтинга — убран `|| true`.
3. Нет обработки ошибок при загрузке изображения в hero — добавлен `onerror` с fallback SVG.
4. В script.js используется `console.error` в продакшн-коде — заменён на `showToast`.
5. Отсутствует meta description для SEO/доступности — добавлен.
6. Добавлены alt-тексты для всех изображений.
7. Улучшена доступность: aria-атрибуты, семантическая разметка, клавиатурная навигация для звёзд.

## Исправления
- Все файлы завершены до полного рабочего состояния.
- Удалён placeholder.com из README.
- `console.error` заменён на пользовательские уведомления через `showToast`.
- Добавлен `meta description`.
- Добавлен `onerror` для hero-изображения с fallback.
- В CI.yml убран `|| true`.
- Добавлены aria-атрибуты для бургер-меню и звёздного рейтинга.
- Добавлена функция `escapeHtml` для защиты от XSS.
- Улучшена валидация формы.

GitHub Pages — только статика/SPA. Все файлы готовы к деплою.
